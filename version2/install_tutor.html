1.开启游戏

  1. 1系统设定：调整音乐及音效。
  2. 2游戏帮助：介绍本游戏。
  3. 3开始游戏。
  4. 4兑换点数：兑换方式。
  5. 5个人资料：可切换玩家身分及观看玩家信息。
  6. 6排行榜：观看排名。

2.选择角色

您可以选择要使用富二代、黑道大哥或是酒店红牌做为您的游戏角色

3.开始游戏

现在，就开始努力地去得分吧!

4.丢酒瓶

按压右下角按钮可丢酒瓶，击败障碍物（人），如服务员、公安等，避免消耗能量。

5.跳跃

按压左下角按钮可跳跃，进行得分或闪避障碍物等动作

6.二段跳跃

在空中点击左下角按钮，可以施展腾空跳跃，如果连续点击，还有短暂旋转飞行的能力（需要消耗能量）。

7.游戏暫停

按压左上角暂停按钮，即暂停游戏，可选择重新开始游戏或继续游戏。

8.能量消耗

碰触障碍物，如服务员、公安、香蕉皮等，均会消耗能量（左上角黄色能量值），能量消耗完毕即无法进行飞行动作。

9.无敌状态

碰触到无敌药水（红圈处），角色周围会有保护罩，碰触任何障碍物在短时间内均不会消耗能量。

10.游戏结束

碰触到公安或炸弹，即游戏结束。

11.送出分数

游戏结束后可送出分数，累积分数可换奖品。

12.加入会员

尚未加入会员请填写数据，以累积分数。
